重启

发表于 2017-07-24 | | 浏览量

Author: dongmodao

作为自控力不行的人，总是推翻自己作下的一个个决定。

自从放假以来，专心用于学习的时间很少很少。虽然明知道自己的能力还是有许多欠缺的地方，当时作为懒散惯了的人来说，在“无事可做”的假期，实在是很难摆脱怠惰的日常。记得看到一个叫做尹梅妮的同学说：「当没有了自制力，自由就会滥化成懒散」。就目前的情况而言，实在是不能再多的认同这句话。

今天想着好久都没有出来认认真真的学习了(平时在宿舍的学习效率简直是负数)，决定出来敲敲代码。发现手感有些生疏，思绪变得阻塞。太久没有使用大脑进行思考，久而久之，用进废退。刷了在 github 上面 fork 的一个大学生面试的库的一篇文章，想着自己还是差的远着呢，所以就有了这篇随笔的出现。也就是说，在接下来的假期时间里，我的“自由”生活即将告一段落。开始新的一段为自己加油增值的一个历程。

我作为一个正常人，不能否认自己有着种种不足之处，种种优缺点。类似与不能坚持自己以往正确的观点，总是犯同样的错误，lazy 等，这是我不能避免的，但是我也不能假装我是完美的，人总是要能正视自己的。我虽然懒，但是，在必要的时候，我还是能够控制自己，尽可能不让自己偏离正确的航向太远。也正是这样，我不得不承认最近一段时间以来自己在行为和思想上的懒惰。但从现在开始，我必须为自己这段时间的所做所为付出代价。

2017 年已经过去一半多了，从开学初的新学期计划来，完成度并不是太高，但不可否认的说，也不是一事无成。就学习语言这个栏目来说，除了 English 这个选项没有太好的实现之外，其他的语言在整段时间来说，或多或少都有不少的收获，整体上有了较大的提高。英语的学习则仅限与阅读一些英文网站，挑战一下，就没有更多的深入，这是这半年比较遗憾的地方。在掌握 IDE这个栏目来说，基本上都有使用到，但是熟练度还有待提高，尤其是 IJ 使用少，了解就少。在另外这个栏目上，除了最后一条，其他的基本上一定程度上能够完成，虽然距离既定的目标也是有一些差距。至于最后一条，「寻找公司，了解社会运行」，目前没有太多涉及，一方面是自己心理上还是有着过不去的坎，另一方面是我也知道我目前的水平实在怕是没公司会收。。。可能有点不自信的说。

在接下来的时间里，把更多的心思投入到写代码进行练习中，学习基础的东西，熟能生巧，稳扎稳打，把以前忽略或者没仔细了解的东西再过一遍，争取有更大的进步。另外，今天刷的这篇 ABC: Always Be Coding 是一篇深度好文，多看几遍，相信每回都会有收获。

就目前的情况而言，重新定义一下自己要稳扎稳打的几门语言，提高熟练度，排名不分先后。其他语言可以稍作了解，目前可能不会进行太过深度的学习：

  • C++
  • Java
  • C#
  • Python

至于学习的基本内容，基本包括但不限于：数据结构、树、图、排序等。总体内容可以根据 coding-interview-university 这个标准走。

今天言尽于此，今天和明天和以后，加油！
