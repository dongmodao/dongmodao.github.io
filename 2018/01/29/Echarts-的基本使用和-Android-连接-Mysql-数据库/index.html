使用 Echarts

Echarts 是什么

Echarts 是由百度公司开发的一个轻量级的可以轻松实现直观、可交互、高度自定义图表和可视化的 canvas 库，基于 zrender 和轻量级的 JavaScript 语言。其官方网址为 Echarts，在这里就不再多说，官网比我说的更加详细。
